nvm install 20 -> instala a versão 18
nvm install --lts -> instala a versão logn-term support (versão recomendada)
nvm list
nvm use 18 -> use a 18 já está instalado na minha maquina
nvm alias default 18 -> define 18 -> define a versão padrão do Node.js para ser a 18

criar .nvmrc - com a versão desejada e nvm use para ativar aquela versão
node -v > .nvmrc -> salva a versão apropriada para aquele projeto
